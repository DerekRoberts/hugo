# About Metacurious

Welcome to Metacurious, a space dedicated to exploring ideas, creativity, and knowledge.

This site has been migrated from Wix to Hugo and is now hosted on GitHub Pages, providing better performance, version control, and easier content management.

## What is Metacurious?

Metacurious is a platform for sharing thoughts, insights, and discoveries across various topics that spark curiosity and inspire learning.

## About This Site

This website is built with [Hugo](https://gohugo.io/), a fast and flexible static site generator, and hosted on [GitHub Pages](https://pages.github.com/).
